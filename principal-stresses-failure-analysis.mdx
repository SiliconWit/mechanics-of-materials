<BionicText method="advanced" intensity="medium" class="highlight">

## üìö Fundamental Theory: Principal Stress Analysis

### The General Stress State

At any point in a stressed body, the stress state can be described by:

**2D Stress Components:**
- Normal stresses: œÉ_x, œÉ_y
- Shear stress: œÑ_xy (= œÑ_yx)

**3D Stress Components:**
- Normal stresses: œÉ_x, œÉ_y, œÉ_z
- Shear stresses: œÑ_xy, œÑ_yz, œÑ_zx

### Principal Stress Theory

<Tabs>
  <TabItem label="Principal Stress Equations">
    
    <Card title="üî± Principal Stress Equations" icon="document">
    **For 2D stress state:**
    $$\sigma_{1,2} = \frac{\sigma_x + \sigma_y}{2} \pm \sqrt{\left(\frac{\sigma_x - \sigma_y}{2}\right)^2 + \tau_{xy}^2}$$
    
    **Principal angles:**
    $$\tan(2\theta_p) = \frac{2\tau_{xy}}{\sigma_x - \sigma_y}$$
    
    **Where:**
    - $\sigma_1, \sigma_2$ = Principal stresses (Pa)
    - $\sigma_x, \sigma_y$ = Normal stresses on coordinate planes (Pa)
    - $\tau_{xy}$ = Shear stress on coordinate planes (Pa)
    - $\theta_p$ = Orientation of principal planes (degrees)
    
    **Physical Meaning:** Principal stresses are the maximum and minimum normal stresses at a point, occurring on planes with zero shear stress.
    </Card>

    
  </TabItem>
  <TabItem label="Maximum Shear Stress">
    
    <Card title="üåÄ Maximum Shear Stress Analysis" icon="document">
    **In-plane maximum shear:**
    $$\tau_{max} = \frac{\sigma_1 - \sigma_2}{2}$$
    
    **Maximum shear planes:**
    $$\theta_s = \theta_p \pm 45¬∞$$
    
    **Associated normal stress:**
    $$\sigma_{avg} = \frac{\sigma_1 + \sigma_2}{2} = \frac{\sigma_x + \sigma_y}{2}$$
    
    **Physical Meaning:** Maximum shear stress occurs on planes oriented 45¬∞ from the principal planes, with the average normal stress acting on those planes.
    </Card>
    
  </TabItem>

### Mohr's Circle Construction

Mohr's circle provides a graphical method for stress transformation:

<Card title="‚≠ï Mohr's Circle Parameters" icon="document">
**Circle Center:**
$$C = \left(\frac{\sigma_x + \sigma_y}{2}, 0\right)$$

**Circle Radius:**
$$R = \sqrt{\left(\frac{\sigma_x - \sigma_y}{2}\right)^2 + \tau_{xy}^2}$$

**Physical Meaning:** Mohr's circle graphically represents all possible stress states at a point as the coordinate system is rotated. The circle's diameter equals the difference between principal stresses.
</Card>

**Key Points on Circle:**
- Principal stresses: œÉ‚ÇÅ and œÉ‚ÇÇ (intersections with œÉ-axis)
- Maximum shear: œÑ_max (top and bottom of circle)
- Any stress state: Point on circle circumference

:::tip[Mohr's Circle Applications]
1. **Find principal stresses** graphically
2. **Determine stress at any angle** of rotation
3. **Visualize stress transformation** relationships
4. **Identify critical orientations** for failure analysis
:::

## üîß Application: Universal Joint Pin Analysis

Let's analyze a critical universal joint pin under complex loading.

:::note[Problem Statement]
**Heavy-Duty Robotic Universal Joint Pin Analysis**

**What we need to determine:**

**Key Question:** Can a 25 mm diameter universal joint pin safely handle 500 N¬∑m torque and 200 N¬∑m bending with SF = 4, and which failure theory governs the design?
:::

<hr />

**System Parameters:**
- Heavy-duty robotic universal joint pin (critical drive component)
- Pin diameter: 25 mm (solid circular cross-section)
- Material: Steel 4340 (**`œÉ_yield`** = 470 MPa, œÉ_ultimate = 745 MPa)
- Applied torque: **`T`** = 500 N¬∑m
- Bending moment: **`M`** = 200 N¬∑m (from joint angulation)
- Contact pressure: 50 MPa (localized bearing load)
- **`Safety factor`**: 4 (industrial environment)

<hr />

### Step 1: Calculate Individual Stress Components

<details>
<summary>**Click to reveal individual stress component calculations**</summary>

<Steps>

1. **Cross-sectional properties:**
   
   $$A = \frac{\pi d^2}{4} = \frac{\pi (25)^2}{4} = 491 \text{ mm}^2$$
   $$I = \frac{\pi d^4}{64} = \frac{\pi (25)^4}{64} = 19,175 \text{ mm}^4$$
   $$J = \frac{\pi d^4}{32} = \frac{\pi (25)^4}{32} = 38,350 \text{ mm}^4$$

2. **Bending stress at surface:**
   
   $$\sigma_x = \frac{Mc}{I} = \frac{200 \times 10^3 \times 12.5}{19,175} = 130.4 \text{ MPa}$$

3. **Torsional shear stress:**
   
   $$\tau_{xy} = \frac{Tr}{J} = \frac{500 \times 10^3 \times 12.5}{38,350} = 163.2 \text{ MPa}$$

4. **Contact stress (localized bearing load):**
   
   $$\sigma_y = -50 \text{ MPa (compressive)}$$

</Steps>

</details>

### Step 2: Principal Stress Analysis Using Mohr's Circle

<details>
<summary>**Click to reveal Mohr's circle and principal stress calculations**</summary>

<Steps>

1. **Stress state at critical point:**
   
   - œÉ_x = 130.4 MPa (tension from bending)
   - œÉ_y = -50 MPa (compression from contact)
   - œÑ_xy = 163.2 MPa (from torsion)

2. **Mohr's circle parameters:**
   
   $$\sigma_{avg} = \frac{\sigma_x + \sigma_y}{2} = \frac{130.4 + (-50)}{2} = 40.2 \text{ MPa}$$
   
   $$R = \sqrt{\left(\frac{\sigma_x - \sigma_y}{2}\right)^2 + \tau_{xy}^2} = \sqrt{\left(\frac{180.4}{2}\right)^2 + 163.2^2} = 186.4 \text{ MPa}$$

3. **Principal stresses:**
   
   $$\sigma_1 = \sigma_{avg} + R = 40.2 + 186.4 = 226.6 \text{ MPa}$$
   $$\sigma_2 = \sigma_{avg} - R = 40.2 - 186.4 = -146.2 \text{ MPa}$$

4. **Maximum shear stress:**
   
   $$\tau_{max} = \frac{\sigma_1 - \sigma_2}{2} = \frac{226.6 - (-146.2)}{2} = 186.4 \text{ MPa}$$

</Steps>

</details>

### Step 3: Apply Failure Theories and Design Assessment

<details>
<summary>**Click to reveal failure theory analysis**</summary>

<Steps>

1. **Maximum Normal Stress Theory:**
   
   $$\sigma_1 = 226.6 \text{ MPa} \text{ vs } \frac{\sigma_{yield}}{SF} = \frac{470}{4} = 117.5 \text{ MPa}$$
   **Result:** Fails criterion ‚ùå (226.6 > 117.5 MPa)

2. **Maximum Shear Stress Theory (Tresca):**
   
   $$\tau_{max} = 186.4 \text{ MPa} \text{ vs } \frac{\sigma_{yield}}{2 \cdot SF} = \frac{470}{8} = 58.75 \text{ MPa}$$
   **Result:** Fails criterion ‚ùå (186.4 > 58.75 MPa)

3. **von Mises Equivalent Stress:**
   
   $$\sigma_{eq} = \sqrt{\sigma_1^2 - \sigma_1\sigma_2 + \sigma_2^2} = \sqrt{226.6^2 - 226.6(-146.2) + (-146.2)^2} = 342.7 \text{ MPa}$$
   
   $$\sigma_{eq} = 342.7 \text{ MPa} \text{ vs } \frac{\sigma_{yield}}{SF} = 117.5 \text{ MPa}$$
   **Result:** Fails criterion ‚ùå (Actual SF = 1.37)

4. **Critical design assessment:**
   
   All failure theories indicate current pin design is inadequate
   Universal joint pin requires immediate redesign to prevent failure

</Steps>

</details>
